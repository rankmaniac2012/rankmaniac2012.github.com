About

Rankmaniac 2012 is a search optimization contest, held annually as a part of Caltech's CS 144: Ideas Behind the Web. We are Kijun Seo, Soonwon Choi, and Euiwoong Lee. This site is awesomely hosted by Github. It's just a static page, but you can still check out the source code here.
